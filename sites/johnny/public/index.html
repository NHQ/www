Auto-Discovery


For me, 'professional development' is keeping track of what is going on in Open Source. I have to know!

I love finding new projects, trying them, and watching them mature. I use Github keep track of dozens of open source projects, and am very resourceful and finding and assessing projects new and old.


A few of my favorite Open Source projects:


  • Node.js - Event-oriented, non-blocking and awesome
  • Redis - Multi-purpose, low-level, and extremely fast database
  • Browserify - Front-end module management for large javascript applications, the Node.js way.
  • Jade - HTML Templating with a Javascript-like syntax (works in the browser too!)
  • Stylus - CSS Preprocessor and Power Up (like SASS and LESS), by the man behind Jade
  • Paper.js - Amazing vector graphics library for HTML Canvas, with an excellent API
  • Timbre.js - Very new, very impressive library for audio synthesis in the browser.
